
# ITSS in Japanese(1)のReactプロジェクト（木2）チーム６
## はじめに
クイズをやれる小さなクイズアプリです。まだまだ未熟なところがたくさんありますが、ご指導のほどよろしくお願いします。
## 起動手順

プロジェクトのフォルダーまで`cd`して
```
$ yarn
$ yarn start
```
を実行すれば起動できるはずです。
